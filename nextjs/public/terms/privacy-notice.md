# سياسة الخصوصية - منصة أحلام

آخر تحديث: 20 مايو 2024

## 1. مقدمة
تلتزم منصة أحلام بحماية خصوصية بياناتكم. توضح هذه السياسة كيفية جمع واستخدام وحماية المعلومات عند استخدامكم لخدماتنا.

## 2. المعلومات التي نجمعها
نحن نجمع المعلومات الضرورية فقط لتقديم تجربة تعليمية متميزة:
- **البيانات الشخصية:** البريد الإلكتروني، الاسم الكامل، وكلمات المرور المشفرة.
- **بيانات المؤسسة:** أسماء المنظمات والفرق التعليمية.
- **محتوى الاختبارات:** الأسئلة، الإجابات، ونتائج الطلاب.
- **البيانات التقنية:** عناوين IP، نوع المتصفح، ومعلومات الجهاز لضمان أمن المنصة.

## 3. كيفية استخدام المعلومات
نستخدم بياناتكم للأغراض التالية:
- إدارة الحسابات وتوفير الوصول للمنصة.
- معالجة وتوليد الاختبارات باستخدام الذكاء الاصطناعي.
- تحسين جودة الخدمات وتطوير أدوات جديدة للمهندسة أحلام.
- التواصل معكم بخصوص تحديثات النظام أو الدعم الفني.

## 4. حماية البيانات
نحن نطبق معايير أمنية صارمة:
- تشفير البيانات أثناء النقل (SSL).
- تشفير مفاتيح API الخاصة بالمستخدمين (AES-256).
- تخزين آمن لنتائج الطلاب عبرRow Level Security (RLS) في Supabase.

## 5. حقوق المستخدم
لديكم الحق في:
- الوصول إلى بياناتكم الشخصية.
- تعديل أو تصحيح المعلومات غير الدقيقة.
- طلب حذف الحساب والبيانات المرتبطة به.

## 6. ملفات تعريف الارتباط (Cookies)
نستخدم ملفات تعريف الارتباط لأغراض تقنية وتأمين جلسات تسجيل الدخول فقط.

## 7. الاتصال بنا
لأي استفسار بخصوص الخصوصية، يمكنكم التواصل مع المهندسة أحلام عبر البريد الإلكتروني الرسمي للمنصة.
